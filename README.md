This is a basic template for a webapp with authentication. The packages that are
being used are:

- Prisma
- Material UI
- Bcrypt
- Next-auth
- Resend for email sending
- UUID
- Zod

The database is PostgreSQL

### How to use

- Go to the app-settings.ts and select your preferences
- Change the .env
- Edit the emails
- Seed the database

Have fun!
